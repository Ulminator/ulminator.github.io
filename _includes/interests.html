Interests

My personal interests include:

  • Machine Learning Infrastructure. Just as enterprises have built out infrastructure that enables engineers to requisition resources for regular applications, there is now an ever growing need for infrastructure that unifies data processing, model training, and deployment for machine learning models across users with varying skillsets with regards to engineering and statistics.
  • Service Mesh. With Kubernetes winning the container orchestration wars, there is now a growing desire for stronger management over microservices with the use of a service mesh architecture that facilitates service discovery, traffic management, observability with dirstributed tracing, and the abstracting of code dealing with security away from the applciation layer. Currently I am learning Golang to help contribute to an internal project at work related service mesh.
  • Computer Networks. Since my time working in the Simbiosys lab at Emory I have been interested in advances in technology that make it feasible to provide cheap, universal internet to developing countries.
